# Image Assets Usage Guide

## Generated Images

### Dashboard Images
- `hero-bg.png` - Main background for dashboard pages
- `stats-bg.png` - Background for statistics cards
- `chart-pattern.png` - Pattern for chart backgrounds

### Fund Categories
- `equity.png` - Equity funds visualization
- `debt.png` - Debt funds visualization
- `hybrid.png` - Hybrid funds visualization
- `sectoral.png` - Sectoral funds visualization
- `international.png` - International funds visualization

### Company Assets
- `logo-full.png` - Full company logo
- `logo-icon.png` - Icon version for sidebar
- `watermark.png` - Watermark for documents

### Partner Logos
- `amfi.png` - AMFI regulatory body
- `sebi.png` - SEBI regulatory body
- `icici.png` - ICICI Bank
- `hdfc.png` - HDFC Bank
- `sbi.png` - State Bank of India
- `axis.png` - Axis Bank
- `kotak.png` - Kotak Mahindra Bank

### User Images
- `default-avatar.png` - Default user profile picture
- `profile-bg.jpg` - User profile background

### Misc Images
- `coming-soon.png` - Coming soon placeholder
- `error-404.png` - 404 error page image
- `success-check.png` - Success confirmation
